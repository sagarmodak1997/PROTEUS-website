2026 · packaging

Single proteus namespace and console script

PROTEUS now installs as one importable package: import proteus exposes the public API (proteus.integrate, proteus.compute_cross_section, and more), and a proteus console script launches the menu (proteus), terminal UI (proteus -t), and web UI (proteus -w). Verify an install with proteus --check. The previous flat module layout and the old script-based entry point are gone.

2026-06-19 · v1.0.0

PROTEUS v1.0.0 — first public release

After a year of development at IISc Bangalore, PROTEUS reaches its first stable release. All planned features are implemented: automatic backend dispatcher, differentiable cross sections, SMEFT operators, hadronic PDFs via PDFFlow, RAMBO phase space, automatic unweighting, UQ ensembles, multi-GPU orchestration, and ONNX export. 246 tests pass on the reference hardware.
